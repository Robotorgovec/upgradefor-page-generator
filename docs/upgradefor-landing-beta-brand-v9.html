Технологическая база

Frontend

Чистый HTML/CSS без внешних зависимостей. Адаптивная сетка, векторные иконки, доступность.

Backend

WordPress + кастомный шаблон для публикаций и токенов. API для синхронизации.

CI/CD

GitHub → Actions → Pages (preview, noindex) → WP. Трассировка изменений через build-version.

Каталог и публикации

Каталог решений

Единая структура карточек, фильтры и теги. Публикуем верифицированные данные.

Пилоты и партнёры

Формы заявки, статус прохождения, публичные результаты.

Сообщество

Профили, сообщения, подборки — после запуска основных модулей.

Роадмап (ближайшие шаги)

Этап 1
Запуск публичного каталога и формы онбординга для авторов.
Этап 2
Ассистент публикации (подсказки качества, авто‑саммари, перевод).
Этап 3
Пилоты и матчинг задач/решений, партнёрские кабинеты.
