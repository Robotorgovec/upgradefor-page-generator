# Визуальный аватар для AI-социальной сети

**Цель:** создать уникальный аватар-образ человека, сочетающий все цвета радуги и плавно переливающийся.

**Требования:**
- Спектр: полный RGB/HSV, мягкие градиенты.
- Эффект: переливание (CSS/WebGL shader, предпочтительно аппаратное).
- Носители: лого ядра ИИ, аватар профиля, обложки, onboarding.
- Адаптация: тёмная/светлая тема, SVG/PNG, fallback без анимации.
- Дальше: анимированный вариант (shader) и Lottie для мобильных.
