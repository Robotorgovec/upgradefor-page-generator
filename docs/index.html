UpgradeFor Page Generator Preview

Эта директория предназначена для предпросмотра статических шаблонов UpgradeFor. Для каждого шаблона есть отдельный файл, который можно открыть напрямую.

  • landing-v1.html – базовый шаблон лендинга.
  • landing-beta-brand-v8.html – полная дизайнерская версия V8.

Используйте эти файлы только для визуального тестирования; они не должны индексироваться поисковыми системами.
